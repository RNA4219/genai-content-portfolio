# ニセステージ (Suno オリジナル曲)

「レプリカ」をテーマにした、日本語オリジナル曲の制作ログ兼ポートフォリオです。  
AI作曲サービス Suno を使用しつつ、歌詞とコンセプト設計は人力で行っています。
試行回数を増やせば理想的な歌唱になると思いますが、ここまでとしています。

- テーマ: 見られることでしか存在を確かめられない「私」と、レプリカとして生きる選択
- 想定ポジション: 深夜アニメの ED 曲
- 想定ボーカル像: 芯のある透明感のある女性ボーカル

`lyrics_ja.md` に全文を掲載しています。


## Suno での生成条件（公開範囲）

### Style（公開版）

ミディアムテンポ（BPM 96〜104）の日本アニメ ED 風。  
ややダークでシリアス寄りだが、サビで少しだけ光が差すエモーショナルなロックバラード。

- BPM: 96〜104 / 4拍子
- 編成: ドラム、エレキベース、エレキギター、ピアノ、パッド系シンセ＋所々ストリングス
- 構成:
  - Aメロ: ピアノ＋シンセ主体で抑えめ
  - Bメロ: ドラムとベースが入り、感情を持ち上げていく
  - サビ: ギターとストリングスを厚くしてダイナミクスを出す
- コード感: マイナー主体だが、サビで一瞬だけメジャー感が見える展開

実際に Suno に入れた文章は、`style_public.txt` に近い内容です。

### Vocal（公開版）

日本人女性ボーカルを想定。

- **芯が強くて透明感のあるハイトーン**
- ピッチは安定していて、ビブラートやフェイクは控えめ
- 地声〜ミックスの切り替えがスムーズで、Aimer や LiSA などのアニソンシンガーの系譜に近いニュアンス
- 感情表現は「がなり」よりも、**押さえた熱さと諦め＋決意の同居**を重視

実際の Vocal プロンプトは一部ノウハウが含まれるため、ここでは抽象化した形のみ公開しています（`vocal_public.txt`）。

## 制作メモ

- 元々作成していた80年代寄りの歌謡ロック的な歌詞からスタート
- 現代アニメ ED レベルまで情報を再構成
- テーマとしてのキーワード:
  - 視線 / 舞台 / 台本 / アドリブ
  - レプリカ / 虚構 / 灰かぶり
- 歌詞は LLM との往復で構造を整理しつつ、最終的なフレーズ調整は人力で行っています
